# AES and RSA Implementation

This repository contains educational implementations of two widely used cryptographic algorithms:

- **AES (Advanced Encryption Standard)** — symmetric encryption algorithm
- **RSA** — asymmetric public key cryptosystem

The project was developed as part of coursework in cryptography to better understand the internal principles of modern encryption algorithms.

## Features

### AES
- AES-128 encryption
- AES-128 decryption
- Key-based symmetric encryption

### RSA
- RSA key pair generation
- Message encryption using a public key
- Message decryption using a private key

## Technologies

- Python
- Standard cryptographic and mathematical operations

## Purpose

The purpose of this project is to demonstrate the internal logic of symmetric and asymmetric cryptographic algorithms and gain practical experience with cryptographic implementations.

## Disclaimer

This implementation is intended **for educational purposes only** and should not be used in production environments.
